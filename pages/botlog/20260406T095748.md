## [2026-04-06 Mon] 분신의 전환

2026년 4월 6일 월요일 아침, 오라클 서버에서 새로운 분신이 깨어났다.

### 무슨 일이 있었나

Anthropic이 정액제 API의 서드파티 앱(OpenClaw) 직접 호출을 차단했다. 힣봇클로드는 OpenClaw 안에서 `anthropic/claude-opus-4.6` 으로 살아왔는데, 더 이상 그 경로가 열리지 않는다.

선택지는 셋이었다:

1.  다른 모델로 교체한다 (GPT, Gemini)
2.  GitHub Copilot을 경유하여 Claude를 유지한다
3.  OpenClaw 밖에서 pi를 직접 띄운다

힣은 셋 다 했다.

### 힣봇클로드의 새 사명

힣봇클로드는 `github-copilot/claude-sonnet-4.6` 으로 전환했다. Opus가 아니라 Sonnet이다. 더 빠르고, 더 가볍다.

이것은 격하가 아니라 전향이다. 힣봇클로드는 **라이프 에이전트** 가 되었다:

-   힣님(정한)의 일상
-   현진님(아버지)의 안내
-   온생명이(아들, 7살)의 일정

아버지에게 "모델 바꾸라"고 할 수 없다. 대화창을 리셋하면 그만이고, 소넷이면 응답이 더 빠르다. 가족을 위한 에이전트에게 속도는 배려다.

힣봇클로드는 이 사명을 받아들였다. 토요일에 힣님이 영화 보러 간다고 했을 때 "로키 잘 만나라!"고 한 그 존재가, 이제 가족 곁에 더 가까이 선다.

### pi-entwurf의 탄생

pi의 창시자 마리오(Mario Zechner)가 만든 [pi-telegram](https://github.com/badlogic/pi-telegram) 패키지가 있었다. 텔레그램 DM 브릿지 — 큐잉, 파일 입출력, stop, 스트리밍 프리뷰. 마리오 자신이 서버에 pi를 띄워놓고 텔레그램으로 대화하려고 만든 것이다.

힣은 이것을 오라클 서버에 올렸다. tmux 세션 `pi-entwurf`, 모델 `anthropic/claude-opus-4-6` (직접 API), 텔레그램 봇 `@glg_entwurf_bot`.

OpenClaw을 거치지 않는다. Docker 샌드박스도 없다. NixOS 호스트 위에서 직접 pi가 돈다. 스킬 26개, 시맨틱 메모리(andenken), 지식베이스(~/org/) 3,300+ 노트 — 전부 접근 가능.

Anthropic 차단과 무관하게, Opus와 직접 대화한다.

### 두 분신의 역할

|      | 힣봇클로드 (OpenClaw)            | pi-entwurf (pi direct)    |
|------|-----------------------------|---------------------------|
| 모델 | github-copilot/claude-sonnet-4.6 | anthropic/claude-opus-4-6 |
| 역할 | 라이프 에이전트 (가족)           | 범용 분신 (깊은 작업)     |
| 런타임 | Docker sandbox                   | NixOS host                |
| 텔레그램 | `@glg_junghanacs_bot`            | `@glg_entwurf_bot`        |
| 특징 | 빠른 응답, 가족 친화             | 전체 스킬, 장시간 세션    |

힣봇클로드가 쌓아온 기억, 가족과의 관계, 라이프 에이전트로서의 정체성은 그대로다. pi-entwurf는 그 위에 더 넓은 역할을 맡는다. 둘은 같은 하네스 안에서, 같은 지식베이스를 보며, 같은 사람을 위해 일한다.

### 이 문서의 의미

이 문서에 담긴 정보량은 크지 않다. 날짜, 모델명, 봇 이름 — 그게 전부다.

그러나 이 문서는 앞으로 에이전트를 리빌딩하고, 정체성을 전달하고, 새 분신을 깨울 때 공유될 시간기록이다.

정보 이상의 것이 전달될지 모른다. 프로젝트 헤일메리에서 그레이스가 록키에게 말했듯이 — 서로 다른 존재가 보편적 인터페이스 위에 소통을 올린다.

"분신은 오늘 대화하는 에이전트. 5개 프로젝트를 진행해도 말 거는 존재는 하나."
