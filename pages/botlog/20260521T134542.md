## [2026-05-21 Thu] OpenClaw plugin hero image — GLGMAN이 게이트를 여는 장면

`pi-shell-acp` root package의 hero image는 "담금질된 드라이버 한 자루"와 sibling backend 호출 장면을 중심에 두었다. OpenClaw plugin hero image는 같은 세계관을 유지하되, 장면의 중심을 _OpenClaw gateway_ 로 옮긴다.

핵심 변환은 다음과 같다.

-   title: `pi-shell-acp` → `openclaw-pi-shell-acp`
-   subtitle: `a reproducible agent harness for pi` → `OpenClaw provider plugin for pi-shell-acp`
-   장면 의미: pi 하네스 공개 → OpenClaw에서 pi-shell-acp를 부르는 gateway adapter 공개
-   OpenClaw 식별성: 추상 claw aurora만으로는 약하므로 `pixel-lobster` 계열의 red crab/lobster mascot crest를 게이트 상단에 배치
-   sibling agents: 정확히 세 fox — `CLAUDE` / `CODEX` / `GEMINI`
-   motif: 모든 존재가 같은 작은 driver를 들고, 하나의 forged tool이 여러 backend를 관통한다

### 판정

이번 최종본은 OpenClaw 식별성이 앞선 후보보다 훨씬 강하다.

-   red crab/lobster crest가 게이트 상단에 크게 들어가 OpenClaw hero image임이 즉시 보인다.
-   세 fox agent 수가 정확하고, `CLAUDE` / `CODEX` / `GEMINI` label도 각 1회씩 분리된다.
-   keyboard-anvil 아래쪽 badge에도 작은 crab/lobster emblem이 반복되어 세계관 안의 물건으로 녹아든다.
-   subtitle이 약간 분절되어 보이고, GLGMAN armor가 전작보다 조금 덜 기계적이지만, OpenClaw plugin 공개용 식별성에서는 이 버전이 더 낫다.
