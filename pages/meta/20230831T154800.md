## <span class="org-hashtag">#조직모드</span>: Org Mode for Emacs

(“Org Mode for Emacs” n.d.) Org Mode for Emacs :

-   Org-mode. Complex so you don’t have to be. A versatile organisational system with immense capabilities.

조직 모드. 복잡하지 않아도 됩니다. 다양한 기능을 갖춘 다재다능한 조직 시스템.

Org Mode: Your life in plain text

조직 모드: 일반 텍스트로 보는 나의 삶

A GNU Emacs major mode for keeping notes, authoring documents, computational notebooks, literate programming, maintaining to-do lists, planning projects, and more — in a fast and effective plain text system.

빠르고 효과적인 일반 텍스트 시스템에서 메모, 문서 작성, 계산 노트북, 프로그래밍, 할 일 목록 관리, 프로젝트 계획 등을 위한 GNU Emacs 의 주요 모드입니다.
