## "근황/소식" 계열 영어 단어들

### 핵심 단어
| 한글 | 영어 | 뉘앙스 |
|------|------|--------|
| 근황 | update, status, whereabouts | 현재 상태/소재 |
| 소식 | news, tidings, word | 정보 전달 |
| 안부 | regards, greetings, how-are-you | 인사 맥락 |
| 현황 | current situation, status quo | 공식적 |
| 동향 | trends, developments, movement | 흐름 맥락 |

### 디지털 가든/PKM 맥락에서 어울리는 것
- *`now`* — "Now Page" 개념 (nownownow.com), 현재 무엇을 하고 있는지
- *`updates`* — 가장 무난
- *`status`* — 상태 기록
- *`log`* — 로그/기록
- *`dispatch`* — 뉴스레터 느낌
- *`chronicle`* — 연대기적 기록

## 메타노트로서 역할 못하는 문제

현재 이 노트의 문제점:

1. *`denote-links` 블록이 비어있음* — `근황|소식` 정규식이 연결된 노트를 못 찾고 있음
2. *`KEYWORDS` 섹션이 실질적으로 비어있음* — 태그/키워드 정의 없음
3. *노트 자체가 콘텐츠 노트처럼 작성됨* — 메타 인덱스가 아닌 개인 에세이

### 메타노트로 살리려면

*옵션 A: `now` 페이지로 전환*
```org
#+title: Now (근황/현황)
#+filetags: :meta:now:
#+description: 현재 무엇을 하고 있는가 — nownownow.com 스타일
```

*옵션 B: `updates` 인덱스로 전환*
```org
#+title: Updates & News (근황·소식 인덱스)
#+filetags: :meta:index:
```
— `denote-links` 블록에 `update|news|근황|소식` 포함 노트들을 자동 수집

*옵션 C: filetag를 `:now:` 또는 `:status:` 추가*
— 다른 노트에서 이 메타노트를 참조할 때 식별성 향상
