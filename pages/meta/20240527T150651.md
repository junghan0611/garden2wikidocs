##### English Summary of Question:

"What is the conceptual difference between 'system' and 'platform'?"

##### Key Differences Between System and Platform:

1.  **정의 (Definition)**
    -   /시스템(System)\*: 상호작용하는 구성요소들의 집합으로 특정 목적을 달성하기 위해 설계된 통합체 (e.g., 운영체제(OS), 생태계(ecosystem))
    -   _플랫폼(Platform)\*: 타 개발자_ 사용자가 구축할 수 있는 _기반 인프라_ (e.g., AWS, Android SDK)

2.  **범위 (Scope)**
    -   시스템: 폐쇄적 또는 독립적 운영 가능 (e.g., 항공관제시스템)
    -   플랫폼: 개방적 확장성 강조 (e.g., iOS 플랫폼에 앱 개발)

3.  **기능적 특성 (Functional Traits)**
    -   시스템: 내부 프로세스 최적화 (e.g., 파일시스템 관리)
    -   플랫폼: 외부 개발을 위한 API/도구 제공 (e.g., TensorFlow 플랫폼)

4.  **사용 사례 (Use Cases)**
    -   시스템 사례: 리눅스 커널, 은행 결제시스템
    -   플랫폼 사례: Kubernetes, WordPress

##### 크로스 체크된 자료:

-   IEEE 정의에 따르면 플랫폼은 "애플리케이션 개발을 위한 프레임워크"로 기술되며[1], 시스템은 "상호 연결된 하드웨어/소프트웨어 구성요소"로 정의됨[2].
-   마틴 파울러는 플랫폼을 "다른 소프트웨어가 실행되는 기반"으로 강조[3].
