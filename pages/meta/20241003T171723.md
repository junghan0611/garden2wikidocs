[2025-01-24 Fri 13:22] 둠이맥스 닥터 패키지 설치 - 문제 해결 닥터 검토 [2025-01-24 Fri 13:28] user 다음 문제를 해결하려고 해 우분투를 기준으로 해당 패키지 설치 방법 알려줘 ```text
Checking for stale elc files in your DOOMDIR...
 > Checking your enabled modules...
   > :lang python
     ! Couldn't find nosetests. Running tests through nose will not work.
   > :lang racket
     ! Couldn't find raco fmt. Formatting will be disabled.
   > :lang rst
     ! Couldn't find rstfmt. Formatting will be disabled.
   > :lang web
     ! Couldn't find tidy. Code formatting in HTML modes will not work.
     ! Couldn't find stylelint. Linting for CSS modes will not work.
     ! Couldn't find js-beautify. Code formatting in JS/CSS/HTML modes will not work.
``` assistant 우분투에서 **nosetests** 를 설치하려면 다음 명령어를 사용하세요: ```bash
# pip install nose
pipx install nose
# sudo apt-get install python3-nose
``` ```text
➜ pipx install nose
  installed package nose 1.3.7, installed using Python 3.12.3
  These apps are now globally available
    - nosetests
    - nosetests-3.4
done! ✨ 🌟 ✨
``` 만약 **pip** 이 설치되어 있지 않다면, 먼저 **pip** 을 설치해야 합니다: ```bash
sudo apt update
sudo apt install python3-pip
``` 그 외에도 **raco fmt**, **rstfmt**, **tidy**, **stylelint**, **js-beautify** 를 설치하려면 각각 다음과 같이 설치할 수 있습니다: - **raco fmt**: Racket을 설치한 후, Racket REPL에서 `(require racket/format)` 를 실행하여 사용할 수 있습니다. Racket 설치는 일반적으로 패키지 매니저를 통해 가능합니다. [래킷](https://notes.junghanacs.com/meta/20230907T134700/) 참고
-   **rstfmt**: pip을 통해 설치 가능합니다.
    ```bash
    # pip install rstfmt
    pipx install rstfmt
    
    ```
-   **tidy**: 우분투의 패키지 매니저를 통해 설치 가능합니다.
    ```bash
    sudo apt install tidy
    ```
-   **stylelint**: npm을 통해 설치 가능합니다.
    ```bash
    npm install -g stylelint
    ```
-   **js-beautify**: npm을 통해 설치 가능합니다.
    ```bash
    npm install -g js-beautify
    ```

npm이 설치되어 있지 않다면, 먼저 **npm** 을 설치해야 합니다:

```bash
# sudo apt install nodejs npm
```
