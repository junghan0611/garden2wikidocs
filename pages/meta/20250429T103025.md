#### @user vscode 에서 이맥스의 tramp와 같은 원격 개발을 영어로 뭐라고 하지? 관련 용어를 정리해줘.

@assistant "Remote Development" 또는 "Remote Editing"이라고 합니다. 관련 용어로는 다음이 있습니다:

-   **Remote Development**: 원격 개발
-   **Remote Editing**: 원격 편집
-   **SSH**: Secure Shell (원격 접속 프로토콜)
-   **Tramp**: Emacs의 원격 파일 편집 도구
-   **VSCode Remote - SSH**: VSCode의 원격 개발 확장 기능
-   **Remote Workspace**: 원격 작업 공간
-   **Remote Server**: 원격 서버
-   **Port Forwarding**: 포트 포워딩
-   **ProxyJump**: SSH 프록시 점프

이맥스의 Tramp와 유사한 기능을 VSCode에서는 "Remote - SSH" 확장을 통해 사용할 수 있습니다.
