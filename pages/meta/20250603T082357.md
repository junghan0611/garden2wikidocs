## KEYWORDS: 세대 연령 연령대 생애주기 코호트

### 핵심 용어 — 한글·영어 대응

| 한글   | 영어                             | 비고         |
|------|--------------------------------|------------|
| 세대   | Generation                       | 가장 넓은 개념 |
| 연령 코호트 | Age Cohort                       | 사회학·통계학 용어 |
| 연령대 | Age Group / Age Bracket          | 10년 단위 묶음 |
| 또래 집단 | Peer Group                       | 교육·발달심리 |
| 생애 주기 | Life Cycle / Life Stage          | 아리스토텔레스적 관점 |
| 연배   | One's age group / Contemporaries | 한국 사회 관계적 용어 |

### Syntopicon 연결 개념

-   **MAN** → 인간의 생애 단계 (유년·청년·장년·노년)
-   **TIME** → 역사적 시간과 세대
-   **EDUCATION** → Paideia 관점의 발달 단계
-   **FAMILY** → 세대 간 관계 (intergenerational)

### 생애 단계별 명칭

| 연령대 | 한글       | 영어                   |
|-----|----------|----------------------|
| 10대 | 청소년, 십대 | Teenager / Adolescent  |
| 20대 | 이십대, 청년 | Twenties / Young Adult |
| 30대 | 삼십대, 장년 초입 | Thirties / Adult       |
| 40대 | 사십대, 중년 | Forties / Middle-aged  |
| 50대 | 오십대, 중장년 | Fifties / Mature Adult |
| 60대+ | 노년, 시니어 | Senior / Elder         |

### Paideia 관점 — 아리스토텔레스의 생애 단계론

-   **유년기** (Childhood) → 습관 형성
-   **청년기** (Youth) → 덕성 교육
-   **장년기** (Maturity) → 실천적 지혜 (phronesis)
-   **노년기** (Old Age) → 관조와 지혜

## 무무에서 올라온 이상한 고리 — "20대"의 이중 의미

[SDF×Zig 유연한 상태머신 — DS 20대 페어링 프로파일러](https://wikidocs.net/382564)

상태머신 노트의 "20대"는 디바이스 20대를 동시에 페어링한다는 기술적 맥락이다. 그런데 제목만 보면 "인간 20대의 페어링"으로 읽힌다. 전혀 무관한 맥락에서 튀어나온 숫자가 인간 생애와 만나는 순간 — 이것이 디지털 가든의 세렌디피티다.

> 무무(無無)에서 확 올라온 이상한 고리. 이것은 상태머신 이야기가 아닐세!

20대 디바이스 페어링 → 20대 인간의 페어링(관계 맺기) → 생애 주기 → 세대론. 연상의 연쇄가 지식그래프의 엣지가 된다.
