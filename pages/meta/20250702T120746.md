1.  **Google Pay (일부 기능)**
    -   _플랫폼_: Android/iOS/ 웹(데스크톱)
    -   _특징_: Flutter로 결제 플로우 UI 통합, 네이티브 모듈과 혼용

2.  **Alibaba Xianyu (앱 + 데스크톱)**
    -   _플랫폼_: iOS/Android/Windows/macOS
    -   _성과_: 단일 코드베이스로 5개 플랫폼 지원, 개발 비용 50% 감소

3.  **Canonical (Ubuntu 설치 프로그램)**
    -   _플랫폼_: Linux 데스크톱 전용
    -   _특이점_: Flutter의 Linux 지원을 활용한 OEM 도구

4.  **Rive (인터랙티브 애니메이션 툴)**
    -   _플랫폼_: 웹 _모바일_ 데스크톱(Windows/macOS)

5.  **Superlist (할일 관리 앱)**
    -   _플랫폼_: iOS/Android/macOS/Windows/ 웹
    -   _아키텍처_: Flutter로 프론트엔드 100% 공유, 백엔드는 Dart+Firebase

#### ⚠️ 주의사항

-   **데스크톱 한계점**:
    -   Windows/macOS는 플러그인 지원이 모바일보다 부족할 수 있음
    -   고성능 그래픽/게임엔 **Flutter + Rust** 조합 권장

&gt; 📌 _검증 방법_: &gt; 1. `flutter create --platforms=android,ios,windows,macos,linux=로 멀티플랫폼 프로젝트 생성 > 2. =flutter run -d windows` 등으로 각 플랫폼 테스트 &gt; 3. `webview_flutter`, `desktop_window` 같은 플러그인으로 플랫폼별 기능 확장
