## 개념 정리

### TTS (Text-to-Speech) 텍스트음성변환 음성합성

-   텍스트 → 음성 출력
-   동의어: 소리내어읽기, 음성합성
-   도구: piper, espeak, EdgeTTS, Coqui

### STT (Speech-to-Text) 음성텍스트변환 음성인식

-   음성 → 텍스트 변환
-   동의어: 음성인식, 받아쓰기, dictation, transcription
-   도구: Whisper, whisper.cpp, Groq API, OpenAI API

### ASR (Automatic Speech Recognition)

-   STT의 기술적 용어
-   한국어 ASR 벤치마크: Return Zero, ClovaSpeech, Whisper large-v3
