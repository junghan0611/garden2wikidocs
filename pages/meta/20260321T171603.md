## 유형(類型) typology

유형(類型): 類(무리 류) + 型(틀 형) — "같은 틀에 속하는 무리"

### 핵심 개념어 지형

| 한글 | 영어              | 뉘앙스             |
|----|-----------------|-----------------|
| 유형 | type              | 가장 일반적. 분류된 하나의 종류 |
| 유형론 | typology          | 유형을 체계적으로 연구하는 학문 |
| 종류 | kind, sort        | 일상적. "같은 종류" |
| 부류 | class, category   | 논리적 묶음. 범주에 가까움 |
| 방식 | mode, manner, way | 어떻게 하는가. 양식 |
| 형식 | form, format      | 겉모양, 틀         |
| 원형 | archetype         | 유형의 극단 이상형 (융) |

### 유형 ↔ 관련 개념 구분

-   **유형 vs. 분류**: 분류는 나누는 행위, 유형은 나눈 결과
-   **유형 vs. 범주**: 범주(category)는 논리적 포함 관계, 유형은 유사성 기반 묶음
-   **유형 vs. 원형**: 원형(archetype)은 유형의 극단 이상형, 현실에 없는 순수 모델
-   **유형 vs. 갈래**: 갈래(branch)는 분기 흐름, 유형은 성질 기반 집합
