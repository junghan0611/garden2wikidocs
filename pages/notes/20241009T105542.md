## dougm/bats-mode

(“Dougm/Bats-Mode” 2024)

-   MacEachern, Doug
-   Emacs mode for editing and running Bats tests
-   [엑서시즘: 코딩 플랫폼 서비스](https://wikidocs.net/381150) 을 테스트해면서 확인했다.
-   test on emacs : bats-run-current-test 실행하면 테스트 동작한다.
-   install

## 인스톨 패키지

```bash

npm i -g awk-language-server
npm uninstall bats # use apt-get below

sudo apt-get install bats bats-assert bats-file bats-support
sudo apt-get install gawk gawk-doc
```

## The Janet Programming Language

(“The Janet Programming Language” n.d.)

Janet is a functional and imperative programming language. It runs on Windows, Linux, macOS, FreeBSD and \*nix.
