[[TIP("이 노트에 대하여")]] default-input-method와 default-transient-input-method의 차이를 중심으로 Emacs 다국어 입력 환경을 정리한다. 시스템 레이아웃 전환 대신 에디터 안에서 입력기를 다루는 이유가 분명해진다. [[/TIP]] The `init.el` settings for multilingual editing (input methods) This sets up Emacs for me to be able to type in Greek while still using Emacs key bindings involving modifier keys (I don't get this if I switch keyboard layouts at the system level). Watch my video about multilingual editing: <https://protesilaos.com/codelog/2023-12-12-emacs-multilingual-editing/>. ```emacs-lisp
(setq default-input-method "korean-hangul")
(setq default-transient-input-method "tex")
``` 이맥스에서 default-input-method와 default-transient-input-method의 차이는? `default-input-method` 와 `default-transient-input-method` 는 Emacs에서 입력 메서드를 설정하는 두 가지 방법입니다. 차이점은 다음과 같습니다: - **default-input-method**: 이 변수는 Emacs 세션 전체에서 기본적으로 사용할 입력 메서드를 지정합니다. 이 입력 메서드는 Emacs를 시작할 때부터 적용되며, 사용자가 명시적으로 다른 입력 메서드로 변경하지 않는 한 계속 사용됩니다. 예를 들어, `set-input-method` 명령을 사용하여 다른 입력 메서드로 변경할 수 있습니다. - **default-transient-input-method**: 이 변수는 특정 상황에서 일시적으로 사용할 입력 메서드를 지정합니다. 예를 들어, `C-u` 와 함께 `set-input-method` 를 사용하면 이 변수에 설정된 입력 메서드가 일시적으로 활성화됩니다. 이는 특정 버퍼나 특정 작업을 수행할 때만 다른 입력 메서드를 사용하고 싶을 때 유용합니다. 이 입력 메서드는 사용자가 다른 입력 메서드로 변경하거나, 버퍼를 닫거나, Emacs를 재시작할 때까지 유지됩니다. 요약하자면: - `default-input-method` 는 Emacs 세션 전체에 걸쳐 기본 입력 메서드를 설정합니다. - `default-transient-input-method` 는 특정 상황에서 일시적으로 사용할 입력 메서드를 설정합니다.
