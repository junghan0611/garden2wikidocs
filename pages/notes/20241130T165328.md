## 2024-12-09

### [SUM]: 블루스카이 이맥스 기다리는 중

#### @user  블루스카이를 이맥스에서 사용할 수 있는 방법을 찾아봐줘

#### @assistant  블루스카이(Bluesky) 서비스를 Emacs에서 사용하는 방법은 현재로서는 제한적입니다. 하지만 몇 가지 접근 방법을 고려해볼 수 있습니다:

1.  **Web Browser in Emacs**: Emacs에는 `eww` 라는 간단한 웹 브 라우저가 내장되어 있습니다. 이를 통해 블루스카이 웹사이트에 접속할 수 있습니다.
    ```elisp
    (eww "https://bsky.app")
    ```

2.  **Emacs Web Apps (EWW) with JavaScript**: 기본 `eww` 는 JavaScript를 지원하지 않기 때문에, JavaScript가 필요한 웹 애플리케이션을 사용하려면 `xwidget-webkit` 을 사용하는 것이 좋습니다. 이는 Emacs 27 이상에서 사용 가능합니다.
    ```elisp
    (xwidget-webkit-browse-url "https://bsky.app")
    ```

3.  **External Browser**: Emacs에서 외부 브라우저를 열어 블루스카이를 사용할 수 있습니다. 예를 들어, `browse-url` 명령어를 사용하면 됩니다.
    ```elisp
    (browse-url "https://bsky.app")
    ```

4.  **API 사용**: 블루스카이가 API를 제공한다면, Emacs Lisp으로 직접 API를 호출하여 사용할 수 있는 인터페이스를 만들 수 있습니다. 하지만 이는 개발 능력과 시간이 필요합니다.

5.  **Third-Party Packages**: 현재 블루스카이와 연동되는 Emacs 패키지는 없지만, 미래에 누군가가 만들 수 있습니다. `elpa` 나 `melpa` 같은 패키지 저장소를 주기적으로 확인해 보세요.

6.  **Emacs Shell**: Emacs 내에서 쉘을 열어(`M-x shell` 또는 `M-x eshell`) 블루스카이의 CLI 도구를 사용할 수도 있습니다. 블루스카이가 CLI 도구를 제공한다면 말이죠.

현재로서는 Emacs 내에서 완벽한 블루스카이 경험을 제공하는 방법은 없지만, 위의 방법들을 통해 어느 정도 접근할 수 있습니다. 블루스카이가 더 많은 개발자 도구와 API를 제공하면, 더 나은 통합이 가능해질 것입니다.
