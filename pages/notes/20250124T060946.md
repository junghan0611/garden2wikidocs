### [2025-01-24 W03](https://wikidocs.net/380394)

#### @user org-latex-compiler 변수에 xelatex, pdflatex 차이가 무엇인가?

@assistant `org-latex-compiler` 변수에서 **XeLaTeX** 와 **PDFLaTeX** 의 차이는 주로 **폰트 사용** 과 **컴파일 속도** 에 있습니다.

-   **XeLaTeX** 는 시스템 폰트를 사용할 수 있으며, OpenType 및 AAT 폰트의 고급 기능을 지원합니다. 이는 다양한 언어와 글꼴을 쉽게 처리할 수 있게 해줍니다[2][4].
-   **PDFLaTeX** 는 컴파일 속도가 빠르고, 마이크로 타이포그래피 기능에 대한 지원이 더 좋습니다. 폰트가 LaTeX에 특화된 폰트로 제한됩니다[4].

따라서, 시스템 폰트나 고급 폰트 기능이 필요한 경우 XeLaTeX를 사용하고, 그렇지 않으면 PDFLaTeX가 더 적합합니다.

#### @user 우분투에 xelatex를 설치하는 방법은?

@assistant 우분투에서 XeLaTeX를 설치하는 방법은 다음과 같습니다:

1.  **TeX Live 설치**: 우분투에서는 TeX Live를 설치해야 합니다. 전체 패키지를 설치하려면 다음 명령어를 사용합니다:
    ```bash
    sudo apt-get install texlive-full
    ```
    그러나 용량이 크기 때문에 필요한 패키지만 설치할 수도 있습니다.

2.  **XeLaTeX 설치**: XeLaTeX는 TeX Live의 일부이므로, `texlive-xetex` 패키지를 설치하면 됩니다:
    ```bash
    sudo apt-get install texlive-xetex
    ```

3.  **한글 지원 추가**: 한글을 사용하려면 추가적인 패키지를 설치해야 합니다:
    ```bash
    sudo apt-get install texlive-lang-cjk
    ```

이렇게 설치하면 우분투에서 XeLaTeX를 사용할 수 있습니다.

#### @user pdflatex를 우분투에 설치하는 방법은?

@assistant 우분투에서 PDFLaTeX를 설치하는 방법은 다음과 같습니다:

1.  **TeX Live 설치**: 우분투에서 TeX Live를 설치합니다. 전체 패키지를 설치하려면 다음 명령어를 사용합니다:
    ```bash
    sudo apt-get install texlive-full
    ```
    그러나 필요한 패키지만 설치하려면 다음 명령어를 사용할 수 있습니다:
    ```bash
    sudo apt-get install texlive-latex-base
    sudo apt-get install texlive-fonts-recommended
    sudo apt-get install texlive-fonts-extra
    sudo apt-get install texlive-latex-extra
    ```

2.  **설치 확인**: 설치가 완료되었는지 확인하려면 다음 명령어를 실행합니다:
    ```bash
    pdflatex --version
    ```

이렇게 설치하면 우분투에서 PDFLaTeX를 사용할 수 있습니다[1][2][3].
