@assistant **\*Eglot-Booster의 동작 원리와 목적**

Eglot-Booster는 Emacs의 Eglot과 같은 LSP 클라이언트의 성능을 향상시키기 위한 도구입니다. 이 도구는 **emacs-lsp-booster** 라는 Rust 기반의 래퍼 프로그램을 사용하여 LSP 서버와의 상호작용을 가속화합니다.

**동작 원리:**

1.  **JSON 메시지 최적화**: LSP 서버에서 전송된 JSON 메시지를 Emacs에서 더 빠르게 처리할 수 있는 elisp 바이트코드로 변환합니다. 이로 인해 JSON 파싱 속도가 약 4배 향상됩니다[2].
2.  **I/O 분리 및 버퍼링**: 읽기와 쓰기를 별도의 스레드로 분리하여 I/O 작업에서 발생할 수 있는 블로킹을 방지합니다[2].

**목적:**

-   Emacs의 LSP 클라이언트(Eglot, lsp-mode 등)의 성능을 향상시키기 위해 개발되었습니다.
-   특히, JSON 파싱과 I/O 작업에서 발생하는 성능 저하를 해결합니다.
-   사용자는 Eglot을 사용하는 방식에 변화가 없으면서도 성능이 향상된 경험을 제공합니다[1][2].
