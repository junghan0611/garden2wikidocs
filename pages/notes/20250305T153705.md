@user 이맥스 iedit에 대해서 알려줘

@assistant **\*iedit** 는 이맥스에서 여러 개의 동일한 텍스트를 동시에 수정할 수 있는 도구입니다. 이 기능은 **iedit-mode** 와 **iedit-rectangle-mode** 로 제공되며, 특히 **iedit-mode** 는 버퍼 내에서 특정 단어를 선택한 후 그 단어와 동일한 모든 발생을 실시간으로 하이라이트하고 수정할 수 있게 합니다[1][2].

**iedit-mode** 의 주요 특징은 다음과 같습니다:

-   **실시간 수정**: 한 곳에서 수정하면 모든 하이라이트된 부분이 자동으로 변경됩니다.
-   **하이라이트**: 모든 일치하는 텍스트가 하이라이트되어 쉽게 식별할 수 있습니다.
-   **범위 제한**: 특정 범위 내에서만 수정할 수 있도록 범위를 좁히거나 넓힐 수 있습니다[2][4].

**설치 방법** 은 이맥스의 내장 패키지 매니저를 통해 MELPA 저장소에서 쉽게 설치할 수 있습니다[2].

**사용법** 은 다음과 같습니다:

1.  **iedit-mode** 활성화: `C-;` 키를 눌러 활성화합니다.
2.  **수정**: 하이라이트된 텍스트 중 하나를 수정하면 모든 하이라이트된 부분이 자동으로 변경됩니다.
3.  **종료**: 다시 `C-;` 를 눌러 종료합니다[2][4].

@user multiplecursors와 차이는?

@assistant **\*iedit** 와 **multiple cursors** 는 모두 여러 위치에서 동시에 텍스트를 수정할 수 있는 기능을 제공하지만, 몇 가지 차이점이 있습니다:

-   **iedit**: 동일한 텍스트를 찾아서 동시에 수정하는 데 중점을 둡니다. 특정 단어를 선택하면 그 단어와 동일한 모든 발생을 하이라이트하고 실시간으로 수정할 수 있습니다[2]. 주로 동일한 텍스트를 대체할 때 유용합니다.

-   **multiple cursors**: 여러 커서를 생성하여 비슷하지만 완전히 동일하지 않은 여러 줄의 텍스트를 동시에 수정할 수 있습니다. 각 커서는 독립적으로 작동하며, 비슷한 형태의 텍스트를 수정할 때 유용합니다[1]. Evil-mode와 통합되어 Evil-multiedit으로도 제공됩니다[3][4].

따라서, **iedit** 는 동일한 텍스트의 대체에 적합하고, **multiple cursors** 는 비슷하지만 다른 텍스트를 수정하는 데 더 적합합니다.
