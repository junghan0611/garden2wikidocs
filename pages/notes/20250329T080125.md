[[TIP("이 노트에 대하여")]] C-u 80 M-x 같은 형태로 universal argument를 전달하는 방법을 간단히 확인한다. 잘 알고 나면 사소하지만 막상 헷갈릴 때 다시 찾게 되는 Emacs 기초 메모다. [[/TIP]] BIBLIOGRAPHY Related Notes History - [2025-03-29 Sat 08:01] [|2025-03-29 Sat 08:01|](https://wikidocs.net/380405.md#h-2025-03-29/)

@user 이맥스에서 M-x 호출 할 때 인자 값으로 80을 넣고 싶다면 universal argument 사용법은?

@assistant Emacs에서 `M-x` 호출 시 인자 값으로 80을 넣고 싶다면, **Universal Argument** 를 사용할 수 있습니다. 방법은 다음과 같습니다:

1.  **Universal Argument** 를 호출합니다: `C-u`
2.  숫자 80을 입력합니다: `80`
3.  명령어를 호출합니다: `M-x`

따라서, 전체 키 입력은 `C-u 80 M-x` 입니다. 이때 `M-x` 대신에 실제로 실행하고자 하는 명령어를 입력하면 됩니다.

예를 들어, 특정 명령어 `my-command` 를 실행하고 싶다면, `C-u 80 M-x my-command` 를 입력합니다.

이렇게 하면 `my-command` 에 인자 값으로 80이 전달됩니다. 다만, 명령어가 인자를 받을 수 있는지 확인해야 합니다.
