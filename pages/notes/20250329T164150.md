<https://github.com/hylang/hyrule>

@assistant Hyrule은 Hy 프로그래밍 언어를 위한 유틸리티 라이브러리입니다. 설치 방법은 다음과 같습니다:

1.  **Python 환경 준비**: Hyrule은 Python 환경에서 동작하므로, Python이 설치되어 있어야 합니다.

2.  **Hyrule 설치**:
    -   **pip** 를 사용하여 설치할 수 있습니다. 터미널에서 다음 명령을 실행하세요:
        ```bash
        pip install hyrule
        ```
    -   또는, GitHub에서 소스를 직접 다운로드하여 설치할 수 있습니다. GitHub에서 소스를 다운로드한 후, `setup.py` 파일이 있는 디렉토리로 이동하여 다음 명령을 실행하세요:
        ```bash
        python setup.py install
        ```

3.  **사용법**: 설치 후, Python 스크립트에서 `import hyrule` 로 Hyrule을 불러와 사용할 수 있습니다. Hy의 경우, `(require hyrule [branch])` 와 같은 방식으로 불러올 수 있습니다[2][4].

Hyrule의 문서는 [Hylang.org](<https://hylang.org/>)에서 확인할 수 있습니다. 테스트는 `pytest` 명령어로 실행할 수 있습니다[2].
