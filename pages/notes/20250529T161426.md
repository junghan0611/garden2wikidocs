## 2025 `nvm` 사용 및 npm 업데이트

[2025-05-29 Thu 16:18]

쿼츠의 경우 npm 버전이 낮아서 설치가 안된다. 다음 확인

가용 버전 확인

```text
nvm ls
```

버전 세팅

```text
nvm use v20.14.0
```

npm 업데이트

```text
npm install -g npm@latest
```
