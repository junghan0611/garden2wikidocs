## Infisical 서비스

@user: infisical 서비스에 대해서 알려줘

@assistant: Infisical은 **시크릿 관리(Secret Management)** 및 **환경 구성(Environment Configuration)** 을 위한 오픈소스/클라우드 서비스입니다. 개발자가 코드나 저장소에 비밀번호, API 키, 토큰 같은 민감한 정보를 직접 넣지 않고, 안전하게 저장·공유·주입할 수 있도록 도와줍니다.

주요 특징:

-   **중앙 집중화된 시크릿 관리**: 여러 프로젝트와 환경(dev/staging/prod)별로 차별화된 환경변수 관리.
-   **보안 저장**: 시크릿은 클라이언트 단에서 암호화 후 서버에 저장, 서버는 실제 평문을 알지 못하는 구조(End-to-End Encryption).
-   **자동 주입(Secret Injection)**: GitHub Actions, Docker, Kubernetes, Vercel 등 CI/CD와 통합해 환경변수를 안전하게 주입.
-   **협업 지원**: 팀 단위로 권한 기반 비밀 정보 공유.
-   **오픈소스 + SaaS 제공**: 직접 호스팅하거나 Infisical 클라우드 서비스 이용 가능.

비슷한 서비스로는 HashiCorp Vault, Doppler, 1Password Secrets Automation 등이 있습니다.
